# IkenFeline - Example Mod

## Usage

1. Clone this repository
2. Copy your entire Ikenfell game installation into the `game` folder
3. Download the latest release of [IkenFeline](https://github.com/Ratismal/IkenFeline/releases), extract into the `game` folder
4. Open `ExampleMod.sln` in Visual Studio
5. Start debugging! You should see `ExampleMod` get loaded

Next steps:
- Replace all instances of `ExampleMod` with the name of your mod (optional)
- Update `manifest.yaml` with correct mod information
- Start hacking!
